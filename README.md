#### Usage

1. install yoman

   ```shell
   npm i -g yo
   ```

   ​

2. install generator-react-bucket

   ```shell
   npm i -g generator-react-bucket
   ```

   ​

3. enter into your word space and build the demo

   ```shell
   yo react-bucket
   ```

   ​

4. run the demo

   ```shell
   npm start
   ```

5. now you can see the demo running at http://127.0.0.1:8000
